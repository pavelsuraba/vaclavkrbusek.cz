# Václav Krbůšek - portfolio
next.js + styled-components
https://www.vaclavkrbusek.cz

### Installation
```sh
$ git clone https://github.com/pavelsuraba/vaclavkrbusek.cz.git
$ yarn
```

### Development
```sh
$ yarn run dev
```

### Build
```sh
$ yarn run build
```

### Static export
```sh
$ yarn run export
```
